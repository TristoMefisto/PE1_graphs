Utility & MRS

$$U = U(x_1, x_2)$$ $$dU = U_1' \, dx_1 + U_2' \, dx_2$$
$$\text{Set } dU = 0 \;\implies\; \frac{dx_2}{dx_1} = -\frac{U_1'}{U_2'} = MRS_{1,2}$$
Slope of the indifference curve: the rate at which the consumer is willing to trade x₂ for x₁ while holding utility constant.

Tax Incidence — Competitive Markets

Equilibrium condition (either side): $Q = S(P) = D(P)$
Consumer-imposed (t on buyers)
$$Q = S(P) = D(P+t)$$
PRE
$\max\, U(x)+y \text{ s.t. } Px+y=Z$
FOC: $P = U_x'$
POST
$\max\, U(x)+y \text{ s.t. } (P+t)x+y=Z$
FOC: $P = U_x' - t$
Producer-imposed (t on sellers)
$$Q = S(P-t) = D(P)$$
PRE
$\max\, \pi(s) = Ps - C(s)$
FOC: $P = C'(s)$
POST
$\max\, \pi(s) = (P-t)s - C(s)$
FOC: $P-t = C'(s)$
$$U_x' - t = C'(s) \quad\equiv\quad C'(s) + t = U_x'$$
Statutory incidence (who the tax is levied on) doesn't determine economic incidence — same equilibrium either way.

Elasticity

$$\varepsilon = \frac{dx}{dy}\cdot\frac{y}{x} \qquad \left(\text{log form: } \varepsilon = \frac{d\log y}{d\log x}\right)$$
Demand
$$\varepsilon_D = \frac{dD}{dQ}\cdot\frac{Q}{D}$$
Supply
$$\varepsilon_S = \frac{dS}{dP}\cdot\frac{P}{S(P)}$$
Elasticity-based incidence (who bears the tax)
Sellers (price drop)
$$\frac{dP}{dt} = \frac{\varepsilon_D}{\varepsilon_S - \varepsilon_D}$$
Buyers (quantity drop)
$$\frac{dQ}{dt} = \frac{\varepsilon_S}{\varepsilon_S - \varepsilon_D}$$
The more inelastic side bears more of the tax — its ε dominates the ratio.

Tax Incidence — Monopoly, Linear Demand

$$P = a - bS$$
$$R = PS = aS - bS^2$$
$$MR = \frac{dR}{dS} = a - 2bS$$
EQ: MR = MC
$$a - 2bS = C'(S) + t$$
Add a to both sides → $2(a-bS) = a + C'(S) + t$ → $2P = a + C'(S) + t$:
$$P = \frac{a + C'(S) + t}{2} \qquad\implies\qquad \frac{dP}{dt} = \frac{1}{2}$$
Pass-through of ½ — identical to the competitive-market benchmark. Linear demand has no curvature, so the markup doesn't react to the tax.

Tax Incidence — Monopoly, Isoelastic Demand

EQ: MR = MC
$$P\left(1 - \frac{1}{\varepsilon_D}\right) = C'(S) + t$$
Pre-tax
$$P = \frac{C'(S)}{1 - 1/\varepsilon_D}$$
Post-tax
$$P = \frac{C'(S) + t}{1 - 1/\varepsilon_D}$$
$$\Delta P = \frac{t}{1 - 1/\varepsilon_D}$$

Specific vs Ad-Valorem Tax (Monopoly)

Specific tax t
$$\pi_t = qx - (c + t)x$$
Ad-valorem rate τ
$$\pi_\tau = \htmlClass{ring-red}{\frac{1}{1+\tau}}\big[qx - (c + \htmlClass{ring-green}{\tau c})x\big] = \htmlClass{ring-red}{\frac{1}{1+\tau}}\,\pi'$$
In monopoly, the ad-valorem tax is equivalent to the combination of:
• a specific tax of value τc
• a profit tax at rate $\gamma = \dfrac{\tau}{1+\tau}$  →  profit net of tax: $(1-\gamma)\pi' = \dfrac{1}{1+\tau}\pi'$
A profit tax has no effect on the firm's optimal Q and pre-tax price p — but it raises revenue.

Harberger General-Equilibrium Model

Two sectors, two mobile factors — 10 equations, 10 unknowns
Production sets
$$X_1 = F_1(K_1, L_1) \qquad X_2 = F_2(K_2, L_2)$$
Full employment
$$K_1 + K_2 = K \qquad L_1 + L_2 = L$$
Mobile factors (price equalization)
$$w = P_1 F_{1L} = P_2 F_{2L} \qquad r = P_1 F_{1K} = P_2 F_{2K}$$
Demand
$$X_1 = X_1(P_1/P_2) \qquad X_2 = X_2(P_1/P_2)$$
Introduce tax dτ on K₂ — solve comparative statics for:
$$\frac{dw}{d\tau},\quad \frac{dr}{d\tau},\quad \frac{dP_1}{d\tau},\quad \frac{dP_2}{d\tau}$$
Tracks how the tax burden splits between K↔L (factor substitution) and Sector 1↔Sector 2 (output substitution).

Excess Burden — Harberger Triangle

Partial equilibrium, quasilinear utility
$$EB = -\frac{1}{2}\frac{dQ}{dt}\,t^2, \qquad dQ = D'(q)\,dq$$
Derivation:
$$EB = -\frac12\,dQ\,dt = -\frac12 D'(q)\,dq\,dt$$ $$= -\frac12 D'(q)\,\frac{\varepsilon_S}{\varepsilon_S-\varepsilon_D}\,dt^2 = -\frac12\,\frac{D}{P}\,\frac{\varepsilon_S\varepsilon_D}{\varepsilon_S-\varepsilon_D}\,dt^2$$
using:
$$dq = \frac{\varepsilon_S}{\varepsilon_S-\varepsilon_D}\,dt$$ $$\varepsilon_D = D'\frac{q}{D} \;\Rightarrow\; D' = \varepsilon_D\frac{D}{P} \;\;(q\approx P)$$

Excess Burden — General Case & MEB

Compensating / Equivalent Variation
$$CV = e(P^1,U^0) - e(P^0,U^0) = e(P^1,U^0) - m$$ $$EV = e(P^1,U^1) - e(P^0,U^1) = m - e(P^0,U^1)$$
Shephard's lemma → bounds on Marshallian CS
$$h(P,U) = \frac{\partial e(P,U)}{\partial P}$$ $$CV = \int_{P^0}^{P^1}\! h(P,U^0)\,dP, \qquad EV = \int_{P^0}^{P^1}\! h(P,U^1)\,dP$$
$$EV \;<\; \text{Marshallian } CS \;<\; CV$$
Excess burden = total welfare loss − tax revenue
$$EB(U^0) = CV - (P^1-P^0)\,h(P^1,U^0)$$ $$EB(U^1) = EV - (P^1-P^0)\,h(P^1,U^1)$$
Marginal Excess Burden (MEB)
$$EB(t) = e(P,U) - e(P+t,U) - t\,h(P+t,U)$$
Not usable empirically (requires the expenditure function e) — approximate instead:
$$MEB \approx EB(t+\delta t) - EB(t) \approx -t\,\delta t\,\frac{dh}{dt} - \frac12\,\frac{dh}{dt}(\delta t)^2$$
Collapses to the Harberger triangle when there's no pre-existing tax.
Still needs h — exploit the Slutsky equation
$$\frac{\partial h(P,U)}{\partial P} = \frac{\partial x(P,m)}{\partial P} + \frac{\partial x(P,m)}{\partial m}\,x(P,m)$$

Ramsey Model — Optimal Commodity Taxation

$$\hat q \hat x \le m + w(T-\ell), \qquad m=0,\ \ q_i = P_i+t_i,\ \ N=2 \text{ goods}$$
w: wage  ·  m: non-wage earnings  ·  T: time endowment  ·  ℓ: leisure. Wage normalized to 1, perfectly elastic labor supply:
$$q_1x_1 + q_2x_2 = T - \ell$$
Uniform tax on all goods, incl. leisure ⟹ equivalent to a lump-sum tax
$$(1+t)(P_1x_1+P_2x_2+\ell) = T$$ $$\Rightarrow\; \left(1-\frac{t}{1+t}\right)T = (1-\gamma)T, \quad \gamma=\frac{t}{1+t}$$
Leaving leisure untaxed ⟹ only distortionary taxation is possible
$$(1+t)(P_1x_1+P_2x_2) = w(T-\ell)$$ $$\Rightarrow\; P_1x_1+P_2x_2 = \frac{w(T-\ell)}{1+t}$$
Inverse Elasticity Rule
Consumer problem
$$\max_{x_1,x_2,\ell} U(x_1,x_2,\ell) \;\text{ s.t. }\; q_1x_1+q_2x_2=T-\ell$$ $$\text{FOC: } U_1' = \lambda q_1, \quad U_\ell' = \lambda$$
Gov. revenue
$$R = q_1x_1+q_2x_2-(P_1x_1+P_2x_2)$$
Planner's problem: max social welfare W(consumer's U), s.t. revenue R
$$\max_{x_1,x_2} \mathcal{L} = U(x_1,x_2,\ell) + \alpha\big(q_1x_1+q_2x_2-R-P_1x_1-P_2x_2\big)$$
$$\tau_i = \frac{t_i}{q_i} = -\frac{1}{\varepsilon_{Di}}\cdot\frac{\alpha-\lambda}{\alpha} = -\frac{k}{\varepsilon_{Di}} \;\ge 0$$ $$k=\frac{\alpha-\lambda}{\alpha}$$
Extended rule — many households, $W=\sum_H \theta_H V_H$
$V_H$: utility of household H  ·  $\theta_H$: social weight
$$\frac{t_i}{q_i} \;\propto\; \frac{1-\bar\theta_i}{\varepsilon_{Di}}$$
Goods with more inelastic demand are taxed more heavily; under the extended rule, $\bar\theta_i$ reflects how much of that good's consumption comes from low-weight (richer) households.

Negative Production Externality Model

$$U(x,y) = U(x) + y - d\,P(x)$$
P(x): pollution  ·  d: marginal damage. Consumers maximize U taking pollution as fixed.
Privately efficient equilibrium
$$\max\, U(x)+y = U(x)+m-Px, \qquad \max\, \pi = Px - C(x)$$
$$\underbrace{U'(x^m) = C'(x^m)}_{PMB\,=\,PMC}$$
Social welfare & socially efficient equilibrium
$$W = \underbrace{U(x)+y}_{\text{consumers}} + \underbrace{(Px-C(x))}_{\text{firms}} - \underbrace{dx}_{\text{pollution}}$$ $$= U(x)+m-C(x)-dx$$
$$\underbrace{U'(x^o) = C'(x^o) + d}_{SMB\,=\,PMC\,+\,MD}$$
Pigouvian tax
$$t = d = MD(x^o) \quad\text{such that}\quad PMC \Rightarrow PMC + t$$
Setting the tax equal to marginal damage at the optimum makes private and social incentives coincide.

General Abatement Framework

Optimum defined as
$$\max_{Q}\, B(Q)-C(Q) \;\Rightarrow\; B'(Q^*)=C'(Q^*)$$
Unregulated
$$Q = 0$$
Standard
Impose Q* directly
Tax
$$P^* = C'(Q^*)$$
Full information: tax and standard are equivalent (both implement Q*).
Cost uncertainty (Weitzman: prices vs quantities)
Regulator doesn't know the firm's true cost θ when setting the standard or the tax; compare expected net benefit under each instrument.
$$\max_Q\, E_\theta\big[B(Q) - C(Q,\theta)\big]$$
$$E_\theta\big[B(Q^*) - C(Q^*,\theta)\big] \;\gtrless\; E_\theta\big[B(Q(p^*)) - C(Q(p^*),\theta)\big]$$
Left side: fixed quantity standard Q* (quantity is certain, cost isn't). Right side: price instrument p*, letting the firm choose Q(p*) in response to realized θ (cost is controlled, quantity floats). Which instrument wins depends on the relative curvature (slopes) of marginal benefit vs marginal cost — steeper MC favors quantity control, steeper MB favors price control.

Public Goods — Private Provision

$$\text{Rival: } X=\sum_H x^H \qquad \text{Non-rival: } G^H = G$$ $$M^H = x^H + g^H \qquad MRT_{G,X}=1,\ \ G=g^1+g^2$$
Utility problem
$$\max\, U^H = U^H(x^H,G) \quad\text{with } U_x^H,U_G^H>0,\ \ U_{xx}^H,U_{GG}^H<0$$
Problem involves strategic interaction (each household's contribution is a best response to the other's):
$$g^1=f^1(g^2),\ \ g^2=f^2(g^1) \;\Rightarrow\; g^{1*}=f^1(g^{2*}),\ \ g^{2*}=f^2(g^{1*})$$
Obtaining the slope of the IC in (g¹,g²) space — individual 1:
$$U^1=U^1(M^1-g^1,\ g^1+g^2)$$ $$dU^1 = -U_x^1\,dg^1 + U_G^1\,dg^1 + U_G^1\,dg^2$$
$$\left.\frac{dg^2}{dg^1}\right|_{\bar U^1} = \frac{U_x^1-U_G^1}{U_G^1}$$
Private optimum (fixed IC, individual best-responds): setting dg²/dg¹=0 ⟹ $U_G^1=U_x^1$ (same for individual 2 by symmetry).
$$U_x^1=U_G^1 \;\Rightarrow\; MRS^1_{G,X} = \frac{U_G^1}{U_x^1} = 1 = MRT_{G,X}$$
Each individual only equates their own MRS to MRT — ignoring the benefit their contribution gives the other. This is the source of free-riding / underprovision.

Samuelson Rule

Pareto efficiency requires tangency between the ICs of both individuals:
$$\left.\frac{dg^2}{dg^1}\right|_{\bar U^1} = \left.\frac{dg^2}{dg^1}\right|_{\bar U^2}$$ $$\frac{U_x^1-U_G^1}{U_G^1} = \frac{U_G^2}{U_x^2-U_G^2}$$
$$\text{leading to:} \quad MRS^1_{G,X} + MRS^2_{G,X} = 1 = MRT_{G,X}$$
The efficient condition sums both individuals' MRS against the MRT — unlike the private-provision result above, which only equates each individual's own MRS. This gap is exactly why private provision under-supplies the public good.

Lindahl Prices

Individualized tax shares (not rates): $\zeta^1,\ \zeta^2,\ \ \zeta^1+\zeta^2=1$
Process:
1. Tax shares are announced.
2. Individuals optimize and reveal quantities $g^1$ and $g^2$:
$$\max\, U^H(x^H,G^H) \;\text{ s.t. } M^H = x^H + \zeta^H G^H$$ $$\text{FOC: } U_x^H = \lambda, \qquad U_G^H = \lambda\zeta^H$$
3. Adjust $\zeta^i$ until $G^1=G^2=G^*$ and $(\zeta^{1*}+\zeta^{2*})G^*=G^*$.
Lindahl equilibrium
$$\zeta^{A*}+\zeta^{B*} = MC(G^*)$$ $$MRS^A_{G,X} + MRS^B_{G,X} = MRT_{G,X}$$
Each individual faces a personalized price ζH for the public good equal to their own marginal benefit — the sum of these personalized prices covers marginal cost, replicating the Samuelson condition without needing a government to know individual preferences (in theory).

Tax-Funded Public Provision — Modified Samuelson Rule

X: private good  ·  G: public good  ·  L: labor, −L: leisure  ·  H identical individuals
Production function
$$F(X,G,L) = pHX + P_G G - HL = 0$$
Individuals maximize
$$V(X,G,L) \;\text{ s.t. } (p+t)X = L-\bar T$$ $$\text{FOC: } U_X = \lambda(p+t), \qquad -U_L=\lambda$$
Government maximizes W = HU under the production constraint
$$\mathcal{L} = HU(X,L,G) - \lambda\big(pHX+P_G G-HL\big)$$
FOC w.r.t. G:
$$\frac{d\mathcal{L}}{dG} = HU_G - \lambda\left(pH\frac{dX}{dG}+P_G-H\frac{dL}{dG}\right) = 0$$
This leads to the Modified Samuelson Rule
$$\sum MRS_{G,-L} = \frac{\lambda}{\alpha}\left(MRT - tH\frac{dX}{dG}\right) \;\gtrless\; MRT$$
$MRT$ = physical marginal rate of transformation. The bracket $\frac{\lambda}{\alpha}\big(MRT - tH\,dX/dG\big)$ is the economic correction term — it diverges from the physical MRT because raising G through taxation distorts the labor/consumption choice, so the shadow cost of public funds differs from the pure production trade-off.
